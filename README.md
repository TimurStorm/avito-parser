# Avito-parser

## Установка:
#### 1) Установить [PyCharm](https://www.jetbrains.com/ru-ru/pycharm/).
#### 2) Открыть проект в PyCharm.
#### 3) Установить пакет poetry через терминал:

```bash 
pip install poetry 
```

#### 4) Установить зависимые пакеты через терминал:
```bash 
poetry install
```
#### 5) Готово!
